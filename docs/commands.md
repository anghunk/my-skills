# 命令说明

## 列出命令

```bash
# 列出所有可用Skills
askills ls

# 或使用别名
askills list
```

输出示例：
```
可用Skills:

  git-commit-message
  auto-vitepress
```

## 查看版本

```bash
askills -v
# 或
askills --version
```

## 升级 askills

```bash
askills update
# 或
askills upgrade
```

## 其他选项

- `-g, --global` - 全局安装/卸载模式
- `-h, --help` - 显示帮助信息
